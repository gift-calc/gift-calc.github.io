Gift Calculator

A smart CLI tool that calculates gift amounts with configurable randomness and relationship-based bias

Installation

🍺 Homebrew

macOS & Linux

$ brew tap david-nossebro/gift-calc
$ brew install gift-calc

📦 NPM

All platforms

$ npm install -g gift-calc

🚀 Install Script

Unix/Linux/macOS

$ curl -fsSL https://raw.githubusercontent.com/david-nossebro/gift-calc/main/install.sh | sh

🪟 PowerShell

Windows

> irm https://raw.githubusercontent.com/david-nossebro/gift-calc/main/install.ps1 | iex

Quick Start

Basic Usage

$ gift-calc
73.24 SEK

For a Close Friend

$ gcalc -f 8 --name "Alice"
89.67 SEK for Alice

Custom Amount & Currency

$ gift-calc -b 100 -c USD
87.35 USD

Setup Configuration

$ gift-calc init-config
Configuration saved! 🎉
